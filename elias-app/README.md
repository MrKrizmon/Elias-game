# 🎮 Элиас — мобильное приложение

## Как получить APK за 5 шагов

### Шаг 1 — Создай аккаунт на GitHub
Зайди на https://github.com и зарегистрируйся (бесплатно).

### Шаг 2 — Создай новый репозиторий
1. Нажми зелёную кнопку **«New»** или **«+»** → **«New repository»**
2. Назови его `elias-game`
3. Выбери **Public**
4. Нажми **«Create repository»**

### Шаг 3 — Загрузи файлы
1. На странице репозитория нажми **«uploading an existing file»**
2. Перетащи ВСЕ файлы и папки из архива `elias-app`
3. Нажми **«Commit changes»**

### Шаг 4 — Запусти сборку APK
1. Перейди во вкладку **«Actions»** (вверху страницы)
2. Слева выбери **«Build Android APK»**
3. Нажми кнопку **«Run workflow»** → **«Run workflow»**
4. Жди ~10-15 минут (будет зелёная галочка)

### Шаг 5 — Скачай APK
1. Нажми на завершённый workflow (зелёная галочка)
2. Внизу страницы найди **«Artifacts»**
3. Скачай **`elias-release.apk`**
4. Отправь другу — пусть установит!

---

## Установка APK на Android

1. Получи файл `elias-release.apk`
2. Открой файл на телефоне
3. Если появится предупреждение — нажми **«Установить из неизвестных источников»** → **Разрешить**
4. Готово! Иконка «Элиас» появится на рабочем столе

## Структура проекта

```
elias-app/
├── App.js                    # Точка входа
├── app.json                  # Настройки Expo
├── package.json
├── src/
│   ├── screens/
│   │   ├── HomeScreen.js     # Главный экран
│   │   ├── RulesScreen.js    # Правила
│   │   ├── WordsScreen.js    # Словарь + поиск + добавление
│   │   ├── SetupScreen.js    # Настройки игры
│   │   └── GameScreen.js     # Сама игра
│   ├── data/
│   │   └── words.js          # 80+ слов по категориям
│   └── utils/
│       ├── theme.js          # Цвета и стили
│       └── storage.js        # Сохранение своих слов
└── .github/
    └── workflows/
        └── build-apk.yml     # Автосборка APK
```
